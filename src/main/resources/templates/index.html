System Rezerwacji Sprzętu Zimowego

Hello!

1. Abstrakcyjna Klasa: Equipment

Reprezentuje ogólne cechy sprzętu, takie jak numer seryjny, stan i typ.

  • Atrybuty:
    • serialNumber: String
    • condition: String
    • type: EquipmentType (enum)
  • Metody:
    • getDetails(): abstrakcyjna metoda do implementacji szczegółów sprzętu.

2. Interfejs: Reservable

Definiuje zachowanie dla rezerwacji sprzętu.

  • Metody:
    • boolean reserve(String customerName, LocalDateTime start, LocalDateTime end)
    • boolean cancelReservation()
    • boolean isAvailable(LocalDateTime start, LocalDateTime end)

3. Enum: EquipmentType

Określa typ sprzętu.

  • SKI
  • SNOWBOARD
  • SKATES

4. Final Class: DatabaseConnection

Wzorzec Singleton, zarządzający połączeniem z bazą danych.

  • Metody:
    • getInstance(): zapewnia jedną instancję połączenia.
    • connect() i disconnect().

5. Dziedziczenie: Klasy Sprzętu

Klasy, które dziedziczą po klasie Equipment, np. Ski i Snowboard.

  • Przykład: Klasa Ski
    • Dodaje specyficzne cechy, np. długość nart.

6. Klasa: Reservation

Obsługuje informacje o rezerwacji.

  • Atrybuty:
    • equipment: Equipment
    • customerName: String
    • status: ReservationStatus (enum)
    • start i end: LocalDateTime

7. Enum: ReservationStatus

Status rezerwacji:

  • ACTIVE
  • CANCELLED
  • COMPLETED

8. Klasa: ReservationManager

Zarządza rezerwacjami i ich stanem.

  • Metody:
    • createReservation(Equipment equipment, String customerName, LocalDateTime start, LocalDateTime end)
    • cancelReservation(Reservation reservation)
    • getActiveReservations()

9. Klasa: Main

Testuje system, symulując procesy rezerwacji, anulowania i zwrotu.

10. Testy Jednostkowe

  • Testowanie metod: reserve() i cancelReservation() (czy zmieniają stan poprawnie).
  • Testowanie EquipmentFactory.createEquipment() (czy tworzy odpowiedni typ sprzętu).
  • Testowanie ReservationManager.getActiveReservations().

11. Scenariusz Testowy w Main

  • Stwórz sprzęt za pomocą fabryki.
  • Zarezerwuj sprzęt na określony czas.
  • Wyświetl aktywne rezerwacje.
  • Anuluj jedną z rezerwacji.
  • Sprawdź dostępność sprzętu.
